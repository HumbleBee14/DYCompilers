# Phase 01 — Foundations

Focus: solidify theory and math prerequisites while building the mental model of a compiler front-end.

## Objectives
- Refresh prerequisite math (sets, graphs, probability) and core data structures.
- Understand formal languages, regular expressions, DFAs/NFAs, and context-free grammars.
- Build hand-crafted lexers/parsers and experiment with parser generators.
- Capture key takeaways and flashcards in `notes/`.
- Complete targeted exercises in `exercises/` before writing larger projects.

## Starter Project — Lexer Playground
- Implement regex-based and hand-coded lexers side by side.
- Visualise token streams and parse trees, storing diagrams in `projects/lexer-playground/docs`.
- Add quick property tests for token boundaries in `tests/`.

## Exit Checklist
- [ ] Can explain the phases of compilation and where each data structure lives.
- [ ] Comfortable deriving FIRST/FOLLOW sets and building LL(1)/LR(0) tables.
- [ ] Have a working lexer prototype and documented learning notes.

Move to Phase 02 once these boxes are realistically ticked.
